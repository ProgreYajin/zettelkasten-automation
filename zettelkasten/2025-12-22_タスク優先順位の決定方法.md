# 

タスクの順番の決め方
タスクを書きだしたら、
1. 3分以内に終わるか　→　終わるならその場で終わらせる
2. そもそもやる必要があるか　→　必要ないなら破棄
3. 誰かに頼めるか　→　頼めるなら頼む
4. プロジェクトか　→　プロジェクトメモに入れる
5. 実行する日が決まっているか　→　決まっているならその日に回す
6. 今週やるか　→　やらないなら来週にまわす
これで残ったのは
3分以上かかる、自分がやるべき、今週やるタスク
この中から、今日は何をやるかを決めていく
まずは期限付きから、期限がついてない場合は作業見込が少ないものから
